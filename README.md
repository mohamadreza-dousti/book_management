# book_management
A python-based project for managing books

-----------------------------------------------

Features
  # Manage books (add, remove, search, show)
  # Advance book search using Regex
  # OOP-Based project
  # Modern interface with CustomTkinter
  # Persistent data storage with SQLite3

  -----------------------------------------------

How to Run
  1.Clone the repository:
  '''Bash
  '''git clone https://github.com/mohamadreza-dousti/book_management.git'''
  2.Navigate to the project folder:
  '''Bash
  '''cd book_management'''
  3.Install requirements:
  '''Bash
  '''pip install -r requirements.txt
  4.Run the application:
  '''Bash
  '''python main.py''' or '''python ui.py'''
